## 💡 Методология расчета APY

### Формула расчета APY за 30 дней

**APY (Annual Percentage Yield)** - годовая процентная доходность, рассчитывается на основе реальных данных за 30 дней:

```
APY (30д) = (Fees_30d / TVL) × (365 / 30) × 100%

Где:
Fees_30d = Volume_30d × (Fee_Tier / 100)
Volume_30d = объем торгов за 30 дней (реальные данные из GeckoTerminal OHLCV)
TVL = Total Value Locked (текущие заблокированные средства в пуле)
Fee_Tier = ставка комиссии пула (0.01%, 0.05%, 0.3%, или 1%)
```

### Пример расчета (WETH / USDC 0.05%):
```
Объем за 24ч: $236,551,502
Объем за 30д (реальный): $7,076,545,065
Комиссия пула: 0.05%
Комиссии за 30д: $7,076,545,065 × 0.0005 = $3,538,273
TVL: $95,747,130
APY (30д): ($3,538,273 / $95,747,130) × (365/30) × 100 = 44.96%
```

### Источники данных для APY (30д)

| Метод | Источник данных | Ключ | Точность |
|-------|----------------|------|----------|
| **GeckoTerminal OHLCV** ✅ | Дневные свечи (30 дней реальных данных) | Не нужен | ⭐⭐⭐⭐⭐ |
| Dune Analytics | Блокчейн SQL запросы | Нужен | ⭐⭐⭐⭐⭐ |
| The Graph | Uniswap v3 subgraph | Нужен | ⭐⭐⭐⭐ |
| Revert Finance | Веб-интерфейс | Не нужен | ⭐⭐⭐⭐ |

**Текущий анализ использует GeckoTerminal OHLCV API** для получения реальных дневных объемов за последние 30 дней каждого пула.

## ⚠️ Риски и рекомендации

### Риски для LP-провайдеров

1. **Непостоянные потери (Impermanent Loss)** - при значительном изменении соотношения цен токенов
2. **Волатильность APY** - доходность может существенно меняться в зависимости от объема торгов
3. **Концентрация ликвидности** - неправильно выбранный диапазон снижает эффективность

### Рекомендации по выбору пула

- **Для консервативных стратегий:** Пулы со стейблкоинами (USDC/USDT, DAI/USDC) - минимальные непостоянные потери
- **Для умеренного риска:** WETH/USDT, WBTC/USDC - ликвидные пулы с высоким объемом
- **Для высокодоходных стратегий:** XAUt/WETH, WLFI/WETH - высокий APY, но повышенная волатильность

### Диверсификация
Рекомендуется распределять средства между несколькими пулами с разными характеристиками для снижения рисков.
